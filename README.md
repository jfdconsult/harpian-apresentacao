# Harpian · AlphaDroid — Apresentação Institucional

Protótipo navegável da apresentação institucional Harpian, em HTML interativo.
Feito para revisão e testes do Diogo.

## ▶️ Como abrir

Acesse o link do **GitHub Pages** (publicado nas configurações do repositório) e comece por:

- **`index.html`** — capa / menu de entrada
- **`narrativa.html`** — início da jornada completa (Parte 1)

Navegue com as **setas ← →** do teclado ou pelos botões na tela.

## 🧭 A jornada (na ordem)

1. **Narrativa** (`narrativa.html`) — manifesto, quem somos, a dor, os pilares
2. **Doutrina** (`doutrina.html`) — filosofia, motor, defesa nas crises, Risk Number
3. **Alinhamento** — lentes do cliente → `coverage-area.html` (matriz Risco × Retorno do **universo em dólar** + alocação)
4. **Consequência** — `lideres.html` (vídeo) → `consequencia.html` (gráfico de 5 linhas, abre nos últimos 10 anos, com toggle para o período completo) → `raiox.html` (raio-X de fechamento)
5. **Ativação** — `terminal-trojan.html` → `confirmacao-conta.html` → `proximos-passos.html`

## 🧪 O que testar (sugestão)

- No **Coverage**, arraste os sliders (alocação no HPC11/HPC22) e veja o Risk Number do dólar descer.
- No **gráfico da Consequência**, aperte ▶ para animar as crises, e troque entre **"10 anos"** e **"Completo"**.
- A alocação que você faz no Coverage é levada para o gráfico e para o Raio-X (via `localStorage`).

## 💬 Como comentar

Abra uma **Issue** neste repositório (aba *Issues*) com print + observação, ou comente direto no commit. Cada tela é um arquivo `.html` independente — fácil de referenciar.
